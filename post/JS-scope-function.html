JavaScript Scope & Function

Pengertian JavaScript Scope dan Function

Scope dan Function adalah konsep penting dalam JavaScript yang berpengaruh pada cara variabel didefinisikan, diakses, dan berinteraksi dalam kode.

1. Scope (Lingkup):
Scope adalah aturan yang mengatur di mana variabel dapat diakses dalam kode.
Dalam JavaScript, ada dua jenis scope utama:
Global Scope: Variabel yang dideklarasikan di luar semua fungsi atau blok kode memiliki cakupan global. Ini berarti variabel tersebut dapat diakses dari mana saja dalam kode, termasuk dari dalam fungsi-fungsi. Contoh: 

            
const globalVar = "Saya variabel global";
function foo() {
  console.log(globalVar); // Akses variabel global
}

Local Scope (Function Scope): Variabel yang dideklarasikan dalam fungsi memiliki cakupan lokal, yang berarti mereka hanya dapat diakses dari dalam fungsi tersebut. Variabel lokal tidak dapat diakses dari luar fungsi. Contoh: 

            
function bar() {
  const localVar = "Saya variabel lokal";
  console.log(localVar); // Akses variabel lokal
}

Selain itu, dalam JavaScript modern, ada juga scope blok (block scope) yang diperkenalkan oleh kata kunci let dan const. Variabel yang dideklarasikan dengan let atau const di dalam blok kode hanya dapat diakses di dalam blok tersebut.

2. Function (Fungsi):
Fungsi adalah blok kode yang dapat digunakan kembali yang mengandung serangkaian pernyataan yang dapat dijalankan. Fungsi digunakan untuk menjalankan tugas tertentu atau mengelompokkan kode tertentu ke dalam unit yang dapat digunakan kembali.
Ada dua jenis fungsi dalam JavaScript:
Fungsi Deklaratif (Function Declaration): Ini adalah cara umum untuk mendefinisikan fungsi. Fungsi ini dideklarasikan dengan kata kunci function diikuti oleh nama fungsi, parameter (jika ada), dan blok kode. Contoh: 

                
function greet(name) {
  console.log(`Halo, ${name}!`);
}

Fungsi Ekspresi (Function Expression):
Fungsi juga dapat didefinisikan sebagai ekspresi yang disimpan dalam variabel. Dalam hal ini, nama fungsi bersifat opsional. Contoh: 

                
const multiply = function(x, y) {
  return x * y;
};

Fungsi dapat menerima argumen, mengembalikan nilai (jika diinginkan), dan digunakan untuk menjalankan berbagai tugas dalam kode JavaScript.
Kombinasi dari scope dan fungsi memungkinkan pengelompokkan variabel dan logika yang berbeda dalam kode JavaScript dan mengontrol di mana variabel tersebut dapat diakses. Hal ini penting untuk memahami kedua konsep ini agar Anda dapat mengembangkan kode JavaScript yang efisien dan dapat dipelihara.
