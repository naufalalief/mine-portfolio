Technical Project Assignment - Personal Website

Pada Technical Project Assignment atau TPA ini, penulis mengembangkan sebuah Personal Website dengan menggunakan HTML, CSS, dan JavaScript serta Website menggunakan Media Query, yang menyebabkan tampilan website menjadi responsive.

Penulis membuat website ini untuk memenuhi tugas Technical Project Assignment yaitu Personal Website. Personal Website ini dibuat dengan menggunakan HTML, CSS, dan JavaScript serta menggunakan Media Query supaya tampilan dari website ini bisa menjadi responsif. Referensi website yang dipakai oleh penulis adalah Personal Website dari Chris Nager. Berikut adalah kriteria penilaian dari TPA Personal Website:

Halaman Website, Bobot: 20%.
Keterangan:
Peserta membuat setidaknya minimal empat atau lebih dari empat halaman website.

Penggunaan Git dan GitHub, Bobot: 20%.
Keterangan:
Peserta menggunakan Git & GitHub dalam proses development secara maksimal dan memanfaatkan fitur commit.

Pemilihan Warna dan Font yang menarik dan sesuai, Bobot: 20%.
Keterangan:
Perpaduan warna dan font yang digunakan sangat menarik dan sesuai.

Pemilihan dan Penerapan Box Model serta Layout, Bobot: 20%.
Keterangan:
Peserta telah memahami dan menggunakan padding serta margin yang sesuai.
Peserta juga menggunakan Flexbox untuk mengatur layout.
Dari sisi desain, peserta mendesain layout dengan sangat baik dan sesuai dengan konteksnya.

Konten Website yang Sesuai, Bobot: 20%.
Keterangan:
Peserta menampilkan konten yang sesuai pada seluruh halaman website.
