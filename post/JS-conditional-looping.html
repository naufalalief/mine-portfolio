JavaScript Conditional & Looping

Pengertian JavaScript Conditional & Looping

Conditional & Looping adalah dua konsep dasar yang digunakan untuk mengendalikan alur eksekusi program.

1.JavaScript Conditional (Kondisional):
Conditional atau kondisi digunakan untuk mengatur aliran program berdasarkan suatu kondisi atau pernyataan. Kondisi ini menghasilkan nilai true atau false. Contoh kondisional utama dalam JavaScript adalah if, else if, dan else. Ini memungkinkan Kita untuk menjalankan blok kode tertentu jika kondisi tertentu terpenuhi. Contoh: 

            
            let nilai = 75;
              if (nilai >= 80) {
                console.log("Nilai A");
              } else if (nilai >= 70) {
                console.log("Nilai B");
              } else {
                console.log("Nilai C");
              }

Selain itu, JavaScript juga memiliki operator kondisional (ternary operator) yang memungkinkan Kita menulis pernyataan kondisional dengan sintaks yang lebih singkat. Contoh: 

            
                let umur = 18;
                let status = umur >= 18 ? "Dewasa" : "Anak-anak";
                console.log(status);

2. JavaScript Looping (Pengulangan):
Looping memungkinkan Kita untuk menjalankan blok kode tertentu beberapa kali. Ini sangat berguna saat Kita ingin melakukan tugas yang sama berulang kali tanpa menulis ulang kode tersebut. Contoh pengulangan utama dalam JavaScript adalah for, while, dan do...while. For loop adalah salah satu jenis pengulangan yang digunakan untuk mengulangi kode sejumlah iterasi tertentu. Contoh: 

                
                for (let i = 0; i < 5; i++) {
                  console.log("Iterasi ke-" + i);
                }

While loop digunakan ketika Kita ingin menjalankan kode selama suatu kondisi terpenuhi. Contoh: 

                
                let angka = 1;
                while (angka <= 5) {
                  console.log("Angka: " + angka);
                  angka++;
                }

Do...while loop adalah varian dari while loop, tetapi blok kode akan dijalankan setidaknya satu kali, bahkan jika kondisi awalnya salah. Contoh: 

                
                let x = 1;
                do {
                    console.log("Nilai x: " + x);
                     x++;
                } while (x <= 5);

Kombinasi kondisional dan pengulangan adalah alat yang kuat dalam pemrograman JavaScript yang memungkinkan Kita membuat program yang responsif dan melakukan tugas yang beragam. Kita dapat menggunakan kondisi untuk memutuskan apakah akan menjalankan pengulangan, dan pengulangan untuk mengulangi kode berdasarkan kondisi tertentu.
